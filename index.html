To perform Deseq analysis, you must upload relevent conditions file.
The file must be csv type that contains a column named 'id' that contains identifiers for each row.
The Deseq analysis will be performed on data uploaded to the relevant heat map.
The data must contains a column named 'id' that contains identifiers for each row.
Important for Deseq analysis the data must contains count values (Integers),
The the number of columns in the count data file (not include id column),
must be the same as the number of rows in the conditions file (not include header).
